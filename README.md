## ğŸš€ Getting Started

### Prerequisites

- **Node.js** 18.x or higher
- **npm** 11.x or higher

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/glix-landing.git
   cd glix-landing
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```

The optimized production build will be generated in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```
